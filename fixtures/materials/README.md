# Material mapping fixtures

`mapping-v1.json` is a schema 1.0 user mapping example. Its colour and PBR values are neutral,
project-authored test data. The referenced PNG is intentionally absent so validation tests can prove
that missing user assets produce diagnostics without committing, uploading, or copying the asset.

No Minecraft texture, model JSON, JAR content, resource-pack content, or sampled colour is stored here.
